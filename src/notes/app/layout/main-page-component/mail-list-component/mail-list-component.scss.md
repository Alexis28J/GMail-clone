

- `.selected` è una classe CSS che viene applicata all'elemento `.email-item` quando l'email è selezionata. 
In questo caso, cambia lo sfondo dell'email per indicare visivamente che è stata selezionata.


- `[class.selected]` è una sintassi Angular che permette di aggiungere o rimuovere dinamicamente la classe CSS `selected` in base al risultato della funzione `isSelected(email)`.


-  `.selected { background-color: rgb(221, 200, 200); }` 
colore di sfondo per l'email selezionata
